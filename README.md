# ECO---BUDDY
Eco Buddy is a Python app that promotes eco-friendly habits through gamification. It follows gamification principles I studied in my undergraduate thesis, using points, levels, streaks, and challenges to motivate sustainable behavior. 

The app allows users to track daily eco-friendly actions, earn points, level up, and gradually improve their lifestyle through small but meaningful changes.

 Features
👤 User registration & login system
📊 Habit tracking and daily logging
🌿 Preloaded eco-friendly habit suggestions
⭐ Points & leveling system
📅 Daily progress tracking (calendar/log)
🔁 Streak system (in advanced version)
🛠️ Admin panel for user management


💡 How It Works
Create a user profile
Select habits that affect the environment
Log your daily actions
Earn points when you improve your behavior
Level up and unlock eco-titles like:
Beginner → Eco Hero → Planet Guardian 🌍

The system uses a local SQLite database to store:

Users
Habits
Daily logs
